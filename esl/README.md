# 基于SystemC TLM平台的XGE-PTPv2软件设计

#### 使用说明
在home目录的.bashrc文件中设置以下环境变量<br>
export SC_SIGNAL_WRITE_CHECK=DISABLE <br>
>source ~/.bashrc <br>
然后执行以下步骤：<br>
>cd /path/to/solution <br>
>rm -rf build && mkdir build && cd build <br>
>cmake .. <br>
>make <br> 
>./ptpv2_tlm <br>
